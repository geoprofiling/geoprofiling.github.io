# Where Do Images Come From? Analyzing Captions to Geographically Profile Vision-Language Datasets

This repository contains the project page for our research on geographical profiling of vision-language datasets.

## Repository Setup

This repository is set up for GitHub Pages hosting.

## Contents

- `index.html` - Main project page
- `*_sampled.csv` - Sampled datasets for each noun category
- `*_images/` - Image directories for each noun category

## Setting Up GitHub Repository

1. **Create the repository on GitHub:**
   - Go to https://github.com/new
   - Repository name: `geoprofile-multimodal-datasets`
   - Username: `abhipsabasu`
   - DO NOT initialize with README, .gitignore, or license
   - Click "Create repository"

2. **Push to GitHub:**
   ```bash
   cd /data3/abhipsa/datacomp/laion-en/project_page
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to: https://github.com/abhipsabasu/geoprofile-multimodal-datasets/settings/pages
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
   - Click "Save"

Your page will be live at: https://abhipsabasu.github.io/geoprofile-multimodal-datasets/
